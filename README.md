# Real Estate Application

## Description
This is a real estate application designed to showcase properties, allow users to filter and search listings, and view detailed information about each property.

## Features
- Property listing display
- Search and filter functionality
- Detailed property views
- Responsive design

## Setup

### Prerequisites
- Node.js (version 14 or higher)
- npm (Node Package Manager)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/RealEstate.git
   cd RealEstate
   ```

2. **Install dependencies for the client:**
   ```bash
   cd client
   npm install
   ```

3. **Install dependencies for the server:**
   ```bash
   cd ../server
   npm install
   ```

4. **Install dependencies for the React Native app:**
   ```bash
   cd ../RealEstateApp
   npm install
   ```

### Running the Application

1. **Start the server:**
   ```bash
   cd server
   npm start
   ```

2. **Start the client (web application):**
   ```bash
   cd client
   npm start
   ```

3. **Run the React Native application:**
   ```bash
   cd RealEstateApp
   npm start
   ```
   Follow the instructions in the terminal to run on an emulator or physical device.

## Project Structure
- `client/`: Contains the web application built with React.
- `server/`: Contains the Node.js backend server.
- `RealEstateApp/`: Contains the React Native mobile application.
- `design/`: Contains design wireframes and assets.
- `scripts/`: Contains utility scripts.
